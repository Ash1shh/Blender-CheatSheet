### :bangbang: Prerequisites
- Install Node JS in your computer <a href='https://nodejs.org/en/'>HERE</a>


<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/Ash1shh/Blender-CheatSheet.git
```

Install dependencies

```bash
  npm install
```
Start the server

```bash
  npm run dev
```
